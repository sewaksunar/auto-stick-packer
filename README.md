Proposed File Structure
This structure is designed to follow the logical transition from theoretical concepts to a technical model setup
.
/Auto-Stick-Packer
│
├── README.md               # Project objectives, team roles, and setup instructions 
├── .gitignore              # Exclude LaTeX build files and temporary simulation data
│
├── /docs                   # Project documentation
│   ├── /proposal           # Stage 1 Proposal (Weeks 1-4)
│   ├── /midterm            # Mid-term Progress Report (Weeks 8-10)
│   ├── /final_report       # Final Technical Report (APA 7th Edition)
│   ├── /presentations      # Slides for oral defenses and progress reviews
│   └── /logs               # Mandatory Weekly Progress Logs
│
├── /research               # Problem formulation and scoping
│   ├── /literature_review  # Systematic literature review results
│   ├── /field_visits       # Documentation of real-world operational challenges 
│   └── /research_gaps      # Identified gaps in current stick-packaging methods
│
├── /design                 # Conceptual and detailed design 
│   ├── /alternatives       # Functional analysis of multiple design options
│   ├── /models             # Detailed CAD and modeling files
│   └── /materials          # Engineering justification for material selection
│
├── /src                    # Technical tool validation and simulation 
│   ├── /simulation         # Kinematic and dynamic simulation scripts
│   ├── /validation         # Iterative cycles of analytical validation
│   └── /optimization       # Simulation data and optimization results
│
└── /assets                 # Images, plots, and figures for reports and presentations
    ├── /images             # Photos from field visits and design iterations
    ├── /plots              # Graphs from simulation results
    └── /figures            # Diagrams for reports and presentations
